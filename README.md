# JGB （金箍棒）

寓意像金箍棒一样可长可短使用自如（依靠插件系统），一件强有力的武器能够支持不同开发系统的渐进式开发工具。

## plugins

- jgb-plugin-babel
- jgb-plugin-css
- jgb-plugin-html
- jgb-plugin-json

## Roadmap

- [x]  支持微信小程序编译 (jgb-preset-weapp)
- [ ]  支持支付宝小程序编译
- [ ]  支持百度小程序编译
- [ ]  支持typescript编译
